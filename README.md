# Green Cloud Analyzer - Frontend

Angular frontend application with Tailwind CSS v3.

## Prerequisites

- Node.js v22.12.0 or higher
- npm

## Installation

```bash
npm install
```

## Development Server

```bash
npm start
```

Navigate to `http://localhost:4200`

## Technologies

- Angular 20.3.10
- Tailwind CSS 3.x
- TypeScript
